# Twitter

Twitter/X MCP. 트윗 게시 및 검색.

**Always confirm before posting.** Show full tweet text and get user approval. 280자 제한. 민감정보(비밀번호, API키, 개인정보) 게시 금지. 스팸성 반복 게시 금지.

## Operations

### Post Tweet
- 텍스트 작성 → 사용자에게 보여주기 → 승인 후 게시
- 280자 초과 시 트리밍 또는 스레드 분할 제안

### Search Tweets
- 검색 문법: `from:username`, `to:username`, `keyword`, `#hashtag`, `keyword lang:ko`
- 결과: 작성자, 텍스트, 날짜 형식으로 정리
